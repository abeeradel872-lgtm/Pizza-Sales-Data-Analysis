## 📌 Project Overview
An end-to-end data analysis project focused on a Pizza Store's sales performance. The goal was to transform raw sales data into actionable business insights using SQL for data extraction and Power BI for interactive visualization.

---

## 🛠️ Tech Stack
| Tool | Purpose |
| :--- | :--- |
| SQL Server | Data Cleaning, KPI Extraction & Business Questions |
| Power BI | Data Modeling, DAX Measures & UI/UX Design |
| Excel | Raw Data Source |

---

## 📈 Key Business Insights
* Peak Hours: Friday evenings show the highest order volume.
* Best Seller: The Classic Deluxe Pizza leads in both quantity and total revenue.
* Low Performer: The Brie Carre Pizza is at the bottom, suggesting a need for marketing or menu changes.
* Order Value: Average order value stands at 38.31, with customers typically buying 2-3 pizzas per order.
